## ğŸ“ Making Changes

### To update your age automatically:
1. Open `src/pages/Home.tsx`
2. Line 9: Make sure birth date is correct
   ```typescript
   const birthDate = new Date('2005-05-21');
   ```
3. The age calculates automatically using `differenceInYears()`

### To change text/translations:
1. Open `src/lib/i18n.tsx`
2. Edit the `translations` object (lines 13-56)
3. Changes apply to both English and Arabic

### To update skills:
1. Open `src/pages/Home.tsx`
2. Edit the `skills` array (lines 13-20)
3. Add/remove skills with icons

### To update social links:
1. Open `src/pages/Home.tsx`
2. Edit the `socials` array (lines 23-54)
3. Update URLs and add/remove platforms
